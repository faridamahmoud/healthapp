# healthapp
It is unity(2022.2.14f1) diet and health mobile application integerated with firebase .The user should be able to create an account (sign up) by entering his email and create password and log into his account using username and password. The user should be able to enter his weight, height, age and specify user’s gender to accurately allow the system to calculate user’s calorie intake. The user should be able to choose food type eaten and system provides that food calorie.Then the system updates the remaining calories. The system checks if the user took calories more than supposed to take, system provides a random exercise to do to lose those calories.
